## Three-Tier Automation Strategy

### Tier 1: High-Volume Automation (800-1000 apps)
**Target ATS**: JazzHR, BambooHR, Recruitee, Manatal, Pinpoint

**Approach**: Full automation
- Auto-fill all fields
- Auto-submit applications
- Minimal human intervention

**Execution**:
- 100-150 apps/day
- 70-80% success rate
- 7-10 days to complete

**Tools**: Playwright browser automation

### Tier 2: Smart Automation (600-800 apps)
**Target ATS**: Lever, Greenhouse, Ashby, Bullhorn, Trakstar

**Approach**: Intelligent automation with human oversight
- Auto-fill with random delays
- Human-like mouse movements
- Session rotation every 10 apps
- Optional manual review before submit

**Execution**:
- 50-75 apps/day
- 50-60% success rate
- 10-15 days to complete

**Tools**: Playwright with stealth mode

### Tier 3: Semi-Manual (200-400 apps)
**Target ATS**: Workday, iCIMS, Taleo, SuccessFactors, Jobvite

**Approach**: Pre-fill only, manual submit
- Auto-fill basic fields
- Manual review required
- Manual submit

**Execution**:
- 30-50 apps/day
- 30-40% success rate
- Use during "break times"

**Tools**: Browser extension or manual + clipboard

## Anti-Detection Tactics

### 1. Rate Limiting
- Max 30 apps/hour per IP
- Max 200 apps/day total
- Use breaks between batches

### 2. Behavioral Randomization
- Random delays (2-8 seconds)
- Random mouse movements
- Random typing speed
- Random scroll patterns

### 3. Session Management
- New browser context every 10 apps
- Clear cookies every 50 apps
- Rotate user agents

### 4. IP Rotation (Optional)
For >150 apps/day:
- Use residential proxies
- Rotate IPs every 20 apps
- Services: Bright Data, Oxylabs, Smartproxy

### 5. Timing
- Apply during human hours (9am-9pm)
- Avoid overnight applications
- Take lunch break (12-1pm)
